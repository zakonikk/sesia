Как пользоваться этим планом

Каждый день = теория (20-40 мин) + практика (решение задач). Отмечай выполненные дни — прогресс сохраняется.

Теория

Смотри короткие видео или читай теорию 20-30 минут. Главное — понять концепцию.

Практика

Решай 8–15 задач в день. Пиши код сам. Тестируй крайние случаи.

Повторение

На следующий день быстро повтори вчерашнюю тему перед новой.

Полезные советы

Как решать задачи

  • 1. Прочитай условие 2 раза
  • 2. Напиши план на бумаге / комментариями
  • 3. Пиши код маленькими частями
  • 4. Тестируй на крайних случаях (0, 1, отрицательные)

Python vs C

Python — проще для старта. C — учит работе с памятью. Решай одни и те же задачи на обеих языках — это лучший способ понять разницу.
